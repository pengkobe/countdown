### Dev env
- 4.4.2
- npm 3.9.3
- antd 1.1.2

### How to run
install antd-init. please make sure node>=4.x
```
npm install antd-init -g
```   
  
then 
```
cd my_countdown
npm start
```  

build & deply
```
 npm run build
```

looking for more : https://ant.design/docs/react/getting-started
